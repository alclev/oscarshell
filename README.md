# Oscar's Shell

## Requirements
```
openai
tk
pywebview
```
## Setup 
**Usage designed specifically for ZSH shell**
```
python3 -m venv .venv
source .venv/bin/activate
pip3 install -r requirements.txt
```
## Usage
First configure your model and api key in `config.json`
**To run:**
`python3 src/main.py`
